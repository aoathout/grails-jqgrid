Grails JQGrid

Grails JqGrid Plugin

This project will provide a grails taglib for the JQGrid JQuery component.

  • I have updated a few things with the project. It now handles standard navigator buttons and also has tags for custom edit/add/delete buttons. There are still many, many features to add, but things should be pretty usuable at this point.
There are two projects for this plugin. One is the plugin code itself, the other is a demo app that shows how to use the taglib.

To run the demo app you must first package the plugin. 
cd grails-jqgrid/jqgrid
grails package-plugin
Then in the demo application you must install the plugin (uninstalling any previous versions) 
cd grails-jqgrid/jqgrid-demo
grails uninstall-plugin jqgrid
grails install-plugin ../jqgrid/grails-jqgrid*.zip
At this point you should be able to launch the application. If you navigate to the AuthorController you can view the list page as a jqgrid.

Dependencies

Grails 1.3.3+

This plugin depends on the latest JQuery-UI plugin (which depends on the latest JQuery plugin).
