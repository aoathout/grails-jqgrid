Grails JQGrid

Grails JqGrid Plugin

I have got things to the point where they are usable for most requirements. There are still things todo like master/detail, etc. but I'll get to them soon. I am in the process of getting the plugin up on the main Grails plugin page and into the main Grails plugin svn so you all that is needed to install things is: 'grails install-plugin jqgrid'. I need to finish the documentation for the main Grails plugin page (examples, release notes, etc.) Once that is done I will update this site with details. Since I prefer Git over SVN I will still maintain the code here. Thanks to everybody that provided feedback. Like always, Please feel free to contribute.

This project will provide a grails taglib for the JQGrid JQuery component.

  • I have updated a few things with the project. It now handles standard navigator buttons and also has tags for custom edit/add/delete buttons. There are still many, many features to add, but things should be pretty usuable at this point.

    I have also updated the demo project so both Author and Book list pages use the jqGrid plugin.
There are two projects for this plugin. One is the plugin code itself, the other is a demo app that shows how to use the taglib.

** You no longer have to build or package the plugin. The demo app now uses inline plugin directives in Config.groovy. **

Dependencies

Grails 1.3.4

This plugin depends on the latest JQuery-UI plugin (which depends on the latest JQuery plugin).
